# 🚀 Деплой Telegram-бота на Railway.app (бесплатно)

## Что такое Railway?
Railway — облачная платформа. Бесплатный тариф даёт **500 часов/месяц** (достаточно для круглосуточной работы одного бота). Сервер в облаке, ничего локального.

---

## Шаг 1 — Создать бота в Telegram

1. Открыть Telegram → найти **@BotFather**
2. Написать `/newbot`
3. Придумать имя (например: `ФраКухниБот`)
4. Придумать username (например: `fraukuhni_bot`)
5. BotFather пришлёт **токен** вида:
   ```
   7412345678:AAHxyz...длинная_строка
   ```
   ⚠️ **Сохранить токен — он понадобится на шаге 4**

---

## Шаг 2 — Добавить бота в групповой чат

1. Открыть нужный групповой чат (откуда приходят лиды)
2. Добавить бота как участника: **Добавить участника → @fraukuhni_bot**
3. Если чат приватный — выдать боту права **читать все сообщения**:
   Настройки группы → Администраторы → Добавить → @fraukuhni_bot

---

## Шаг 3 — Загрузить код на GitHub

1. Зарегистрироваться на [github.com](https://github.com) (если нет аккаунта)
2. Нажать **New repository** → назвать `fraukuhni-bot` → Create
3. Загрузить 4 файла из папки проекта:
   - `bot.py`
   - `requirements.txt`
   - `Procfile`
   - `railway.json`
   - `.gitignore`

   Через браузер: кнопка **"Add file" → "Upload files"** → перетащить все файлы → **Commit changes**

---

## Шаг 4 — Деплой на Railway

1. Зайти на [railway.app](https://railway.app)
2. Нажать **"Start a New Project"**
3. Выбрать **"Deploy from GitHub repo"**
4. Авторизоваться через GitHub → выбрать репозиторий `fraukuhni-bot`
5. Railway автоматически найдёт `Procfile` и начнёт сборку

### Добавить переменную окружения (токен бота):
После создания проекта:
1. Открыть проект → вкладка **Variables**
2. Нажать **"New Variable"**
3. Заполнить:
   - **Name:** `TELEGRAM_TOKEN`
   - **Value:** `7412345678:AAHxyz...` (ваш токен из шага 1)
4. Нажать **Add** → Railway автоматически перезапустит бота

---

## Шаг 5 — Проверка работы

1. В Railway открыть вкладку **Deployments → View Logs**
2. Должна появиться строка:
   ```
   INFO - ✅ БД инициализирована: clients.db
   INFO - 🤖 Бот запущен...
   ```
3. Написать в групповой чат сообщение с тегом `#ФРАУ_КУХНИ`:
   ```
   #ФРАУ_КУХНИ Тест +79991234567 город Москва Бюджет 500 000 р
   ```
4. В логах появится:
   ```
   INFO - 📨 Получено сообщение...
   INFO - ✅ Клиент сохранён id=1
   ```

---

## ⚠️ Важные ограничения Railway (бесплатный тариф)

| Параметр | Значение |
|---|---|
| Часы работы | 500 ч/месяц (~20 дней непрерывно) |
| RAM | 512 МБ |
| CPU | Shared |
| Хранилище | 1 ГБ |
| Сон при неактивности | ❌ Нет (worker не спит) |

> 💡 **Совет:** 500 часов хватит на весь месяц, если бот один. Если нужно больше — Railway Pro стоит $5/месяц с безлимитными часами.

---

## 🔄 Обновление бота

Чтобы обновить код:
1. Изменить файл `bot.py` на GitHub (кнопка карандаша → редактировать → Commit)
2. Railway автоматически пересоберёт и перезапустит бота

---

## 🛠️ Если бот не работает — чеклист

- [ ] Токен скопирован полностью (без пробелов)
- [ ] Бот добавлен в групповой чат как администратор
- [ ] Сообщение содержит тег `#ФРАУ_КУХНИ` (с заглавной буквы)
- [ ] В логах нет красных строк `ERROR`
- [ ] Файл `requirements.txt` загружен на GitHub
